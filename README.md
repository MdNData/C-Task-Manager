# C-Task-Manager (Todo list project in C)

Un semplice gestore di Todo List da riga di comando scritto in C puro, come progetto didattico per imparare la gestione della memoria, le strutture dati e le buone pratiche di programmazione.

## Scopo del progetto

L'obiettivo è costruire un'applicazione console completa partendo da zero, seguendo le pratiche di un programmatore esperto, inclusa la definizione dei requisiti, la gestione delle priorità (MVP) e l'architettura del software.

## Come compilare

Dato che stiamo usando Visual Studio Community:
1. Clonare il repository.
2. Aprire il file `.sln` (soluzione) con Visual Studio.
3. Selezionare la configurazione (es. "Debug" o "Release").
4. Premere F5 per compilare ed eseguire.
